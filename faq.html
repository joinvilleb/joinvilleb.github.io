Frequently Asked Questions




Do I need to have any coding knowledge to use these services? ▼

No! The Basic Setup package is made to guide you through the setup process, even if you're a beginner. With Full Setup, I handle everything for you, so you can focus on your business or portfolio.

What's included in the Full Website Setup package? ▼

The Full Setup package includes everything in the Basic Setup package, plus full website setup handled by me. It includes repository creation, complete customization, deployment, and custom domain setup (up to $30 USD in domain registration fees covered). It also includes ongoing support to ensure your site works perfectly.

What types of websites are these packages suitable for? ▼

These packages are ideal for portfolios, personal blogs, and small business websites. They are made for professionals, freelancers, and small businesses looking to establish an online presence without breaking the bank.

Will my website by mobile-friendly? ▼

Yes, both Basic and Full Web Setup packages include a mobile-friendly site. I'll make sure your site looks great on all devices, including smartphones, tablets, and desktops.

How secure will my website be once it's live? ▼

Security is a priority. I ensure every setup follows basic security practices, and I can offer advice on keeping your website secure over time.

What's included in the Basic Website Setup package? ▼

The Basic Setup package includes step-by-step guidance to set up a GitHub repository, personalized site customization, and one-on-one support to help you get started and understand the setup process.

How long does it take to complete each package? ▼

The Basic Setup package typically takes a few days, depending on your availability and my schedule. The Full Setup package can take up to two weeks, as I handle the entire setup and ensure every aspect is tailored to your requirements.

Do you offer any support after the website is live? ▼

Yes, I provide support for every setup. The Basic Setup is active for 12 months, while the Full Setup is active for 18 months. The Full Setup Package includes extended support to answer any questions and resolve any issues that may arise after launch.

Can I switch from the Basic to Full Setup if I change my mind later? ▼

Absolutely! If you start with the Basic Setup and later decide you'd like the Full Setup, you can upgrade at any time.

What is GitHub, and why do you use it for website setup? ▼

Github is a platform for hosting websites and managing code. It's free to use, secure, and offers version control, making it easy to track and manage changes to your site. I use GitHub to make your site accessible, managable, and easy to update. However, with the Full Setup package, you can choose to host your site on a different platform if you prefer (additional fees will apply).

Will you help me purchase a domain, and is it icluded in the setup? ▼

Yes, the Full Setup package includes custom domain setup, covering up to $30 USD in registration fees. If you already have a domain, I can help connect it to your new website.

Are there any hidden costs after purchasing the package? ▼

No hidden costs! The Basic Setup is all-inclusive, and the Full Setup covers domain fees up to $30 USD. If additional fees apply (like premium hosting), I will discuss options with you transparently.

What if I already have a website? Can you help me improve it? ▼

Yes, I offer personalized mentorship to help those with an existing website improve its functionality, apperance, and performance. You can reach out to discuss your specific needs here.
